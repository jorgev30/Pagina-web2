Objetivo general:

Realizar un estudio de la usabilidad y la arquitectura de sitios web, con el fin de evaluar la efectividad de los contenidos y la practicidad que propone el curso de diseños de sitios web, utilizando el modelo Git y GitHub.

Objetivo específico 1:

ostrar la codificación de una página web mediante la estructura HTML, HTML5 y CSS3 utilizando el modelo GitHub

Objetivo específico 2:

Proponer la mejor manera de estructurar la información para la solución de problemas aplicando técnicas de programación, y así mismo Implementar pautas para el desarrollo de un proyecto web

Objetivo específico 3:

Implementar en un lenguaje de programación nuevas estructuras de datos para la implementación del diseño de un sitio web.
